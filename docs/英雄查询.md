# 抗战英雄事迹查询系统 (War Hero Inquiry System)

## 📖 项目简介

本项目是一个基于 **Yii2 Framework** 开发的 Web 端抗战英雄纪念与查询系统。
系统旨在通过数字化手段铭记历史、致敬英雄。采用了庄重的红色主题设计，实现了对英雄人物的**增删改查 (CRUD)**、**多条件筛选**、**卡片式展示**以及**详情页阅览**功能。

---

## ✨ 核心功能 (Features)

### 1. 英雄列表展示 (Hero Gallery)

* **卡片式布局**：摒弃了传统的表格显示，采用 `Bootstrap Panel` 设计了红底白字的卡片样式，每行显示 4 位英雄，界面整洁美观。
* **响应式设计**：适配不同屏幕尺寸。
* **图片显示**：每张卡片均包含英雄的历史照片（自动适配 `.jpg` 或 `.png`）。

### 2. 多条件搜索 (Advanced Search)

* **支持模糊查询**：可以通过输入英雄“姓名”进行模糊搜索。
* **战区筛选**：实现了按五大战区（东北、华北、华中、华东、华南）进行分类筛选。
* **UI 美化**：使用了 `Flex` 布局重构了搜索栏，使其在页面居中且横向整齐排列。

### 3. 自定义分页 (Custom Pagination)

* **后端分页**：在 Model 层设置了 `pageSize => 4`，每页仅显示 4 个数据，避免页面过长。
* **前端样式优化**：重写了分页按钮的 CSS 样式，将其美化为圆角红色方块，并强制居中显示，与整体主题保持一致。

### 4. 英雄详情页 (Hero Details)

* **图文并茂**：详情页采用“左图右文”的档案卡式布局。
* **沉浸式阅读**：去除了 ID 等技术字段，只展示姓名、生卒年、身份、战区、名言及详细事迹。
* **数据丰富**：数据库中已录入包括杨靖宇、张自忠、白求恩（国际友人）、聂耳（艺术家）、陈嘉庚（华侨）等近 20 位涵盖各领域的英雄人物。

## 🏗️ 技术架构 (MVC Structure)

本项目严格遵循 **MVC (Model-View-Controller)** 架构设计模式，满足开发规范：

### 📂 Model (模型层)

* **文件路径**：`frontend/models/HeroSearch.php`
* **功能**：负责处理英雄数据的搜索逻辑、分页配置以及数据验证规则。我们将搜索模型专门放置在前端模型文件夹中，以实现前后端逻辑分离。

### 📂 View (视图层)

* **文件路径**：`frontend/views/hero/index.php`
* **功能**：负责前端页面的渲染。包含自定义的 CSS 样式块、HTML 结构、以及数据的循环展示逻辑。
* **辅助视图**：包含 `_search.php`（搜索栏组件）和 `view.php`（详情页组件）。

### 📂 Controller (控制器层)

* **文件路径**：`frontend/controllers/HeroController.php`
* **功能**：作为系统的“指挥官”，接收用户的请求，调用 Model 查询数据库，并将数据传递给 View 进行显示。

## 🚀 安装与运行 (Installation)

如果您拿到了本项目代码，请按照以下步骤运行：

1. **环境要求**：PHP >= 7.0, MySQL, Web Server (Apache/Nginx)。
2. **代码部署**：将项目文件夹解压至服务器根目录（如 `htdocs`）。
3. **导入数据库 (关键步骤)**：
* 新建数据库名为 `memory_of_war_db`。
* 导入项目附带的 SQL 文件：`memory_of_war_db.sql`。


4. **配置文件**：
* 检查 `common/config/main-local.php`，确保数据库账号密码与您的环境一致。


5. **访问**：
* 打开浏览器访问：`http://localhost/你的项目文件夹/frontend/web/index.php?r=hero/index`

## 📝 数据库概览

系统包含核心数据表 `hero`，字段设计如下：

* `id`: 主键
* `name`: 英雄姓名
* `life_span`: 生卒年
* `identity`: 身份/军衔
* `war_zone`: 所属战区
* `quote`: 名言/座右铭
* `description`: 详细事迹（长文本）
* `photo_path`: 照片文件名
